# file-converter
dibin's file converter. It has great capabilities, completely free. 

[https://dibin17.github.io/file-converter/](https://dibin17.github.io/file-converter/)

* Convert files to PNG, JPG, WEBP, or BMP with the convert tool.
* Make live visual edits with the filter studio tool. Pixelation, stretch and warp included.
* Encode and Decode text and files to Base64.
* Generate FREE qr codes from any link.
* Record live audio, (or attach file) and edit it basically however you want (volume, reverse, robot, etc).
* Hide messages, and show hidden messages, with the steganography tool.
* Strip metadata from your files, see location, info, timestamps, and more.

All this for _free!_ Yessir!!
